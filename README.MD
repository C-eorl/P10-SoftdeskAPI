## I - Installation

#### 1. Cloner le dépôt
```bash
  git clone git@github.com:C-eorl/P10-SoftdeskAPI.git 
  cd P10-SoftdeskAPI
```
#### 2. Créer un environnement virtuel
```bash
    python -m venv venv
    source venv/bin/activate   # sous Windows : venv\Scripts\activate
```
#### 3. Installer les dépendances
```bash
    pip install -r requirements.txt
```
